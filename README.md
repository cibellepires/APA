# APA
🧠 Análise e Projeto de Algoritmos

Repositório dedicado ao estudo de Análise e Projeto de Algoritmos, com implementações e experimentos voltados para compreender eficiência, complexidade e técnicas de desenvolvimento de algoritmos.

O objetivo deste repositório é consolidar conceitos fundamentais como análise de complexidade, estratégias algorítmicas e resolução de problemas clássicos.

🎯 Objetivos

Estudar técnicas de projeto de algoritmos

Compreender análise de complexidade

Implementar algoritmos clássicos

Comparar desempenho entre diferentes abordagens

Desenvolver raciocínio para resolução de problemas computacionais

🛠️ Tecnologias Utilizadas

Linguagem: C / Python
